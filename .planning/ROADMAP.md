# Roadmap - Inventory System Hardening

## Milestone 1: Foundation & Hardening (Current)
- **Phase 1: DB & Logic Alignment** (Migrate to SQL, unify CRUD)
- **Phase 2: Security Deep Dive** (Hashing, CSRF, Prepared Statements)
- **Phase 3: Cleanup & Optimization** (Remove JSON dependency)

## Milestone 2: AI & Analytics Expansion
- **Phase 4: Advanced Forecasting** (Prophet or similar patterns in PHP)
- **Phase 5: Interactive Reporting** (Exporting, PDF generation)

## Milestone 3: Scale & UI
- **Phase 6: Multi-User Roles** (ACL, Admin vs Viewer)
- **Phase 7: Modern UI Rewrite** (Potential migration to SPA or enhanced Bootstrap)
