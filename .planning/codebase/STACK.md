# Tech Stack

## Backend
- **PHP**: Core application logic. Version 7.x or later required. (Procedural style).
- **mysqli**: PHP extension for MySQL database interaction.

## Frontend
- **HTML5/CSS3**: Core structure and basic styling.
- **JavaScript**: Client-side logic for dynamic dashboard features.
- **Bootstrap**: UI framework used for responsive layout and components.

## Storage
- **MySQL**: Relational database used in the root system version (`inventory_db`).
- **JSON**: File-based storage (`data.json`) used in the `Inventory-Management-System/` subfolder.

## Environment
- **Apache/XAMPP**: Recommended local web server environment.
